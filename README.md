# Za-Eng Construction Digital Ecosystem Platform

AI-powered integrated management platform for the construction industry.

## Project Structure

```
├── backend/          # NestJS backend API
├── frontend/         # React + TypeScript frontend
├── mobile/           # React Native app (Phase 2)
├── shared/           # Shared types and utilities
├── docs/             # Documentation
├── infrastructure/   # Docker, CI/CD, deployment configs
└── scripts/          # Utility scripts
```

## Technology Stack

- **Frontend**: React + TypeScript + Tailwind CSS
- **Backend**: Node.js (NestJS)
- **Database**: PostgreSQL + Redis
- **AI/Vector**: OpenAI API / Llama 3 + Pinecone/pgvector
- **File Storage**: AWS S3 / Hetzner Object Storage
- **Hosting**: AWS / Hetzner (South African region)
